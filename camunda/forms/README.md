# Camunda Forms

This directory contains Camunda Forms for human tasks in Tasklist.

## Files

| File | Description |
|------|-------------|
| `manual-assignment.form` | Operator manually assigns provider |
| `escalation-review.form` | Review and resolve escalations |
| `wcf-signature.form` | Customer signs Work Completion Form |

## Editing

Use [Camunda Modeler](https://camunda.com/download/modeler/) to edit these files.
