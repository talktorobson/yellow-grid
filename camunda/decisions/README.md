# DMN Decision Tables

This directory contains DMN decision models.

## Files

| File | Description |
|------|-------------|
| `provider-ranking.dmn` | Score providers based on distance, rating, availability |
| `assignment-mode.dmn` | Determine assignment mode by country (OFFER/AUTO_ACCEPT) |
| `urgency-sla.dmn` | Calculate SLA based on urgency level |
| `escalation-rules.dmn` | Determine when to escalate to operator |

## Editing

Use [Camunda Modeler](https://camunda.com/download/modeler/) to edit these files.
