# Outbound Connectors

This directory contains configuration for Camunda 8 connectors.

## Files

| File | Description |
|------|-------------|
| `kafka.json` | Kafka connector for event publishing |
| `email.json` | Email notification connector |
| `sms.json` | SMS notification connector |

## Usage

Connectors are used in BPMN via Connector Tasks.
See [Camunda Connectors Guide](https://docs.camunda.io/docs/components/connectors/introduction-to-connectors/).
